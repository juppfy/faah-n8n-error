# 🔊 Faah Monitor — v2 (The Instance-Wide God Mode)

> *"Why add an Error Trigger to 50 workflows when I can just watch them all?"*

v2 is the **lazy dev's dream**. Instead of manually editing your n8n workflows, v2 **polls the n8n API** automatically.

If a workflow fails anywhere on your n8n instance, v2 finds it, logs it, makes the workflow name a clickable link, and blasts the FAHHH sound.

No webhooks. No ngrok. Pure unadulterated polling.

---

## 🏃‍♂️ How to Run It

1. **Fork it, Clone it, Bop it**
   ```bash
   git clone https://github.com/juppfy/faah-error-sound.git
   cd faah-error-sound/v2
   ```
   *(Yes, make sure you `cd` into the `v2/` folder!)*

2. **Install & Start**
   ```bash
   npm install
   npm start
   ```

3. **Open the Dashboard**
   Go to **http://localhost:3002** in your browser.
   You will see a fancy "Connect to n8n" screen.

---

## 🔑 Fetching your n8n API Key

v2 needs to talk to n8n. To do that, it needs the sacred texts (an API key).

1. Open your n8n instance in your browser.
2. At the bottom left, click **Settings**.
3. Go to **n8n API** (or User Settings > API, depending on your version).
4. Click **Create an API Key**.
5. Give it a name like "Faah Alarm".
6. Copy that incredibly long string of gibberish.

---

## 🔌 Connecting the Monitor

Go back to your `http://localhost:3002` tab.

1. **n8n Base URL:** Put your n8n link here. (e.g. `http://localhost:5678` or `https://n8n.mycompany.com`)
2. **API Key:** Paste the gibberish from earlier.
3. Hit **Connect**.

> **Is this safe?** Yes! Your API key is saved to your browser's temporary memory (`sessionStorage`) and sent only in the secure POST body to the local server. If you close the tab, the key vanishes. It is never logged or saved to a database.

---

## 💥 What happens next?

1. The app starts polling n8n every 10 seconds (you can change this interval).
2. It waits.
3. It watches.
4. The moment a workflow fails... **FAHHHHH** 🔊
5. The UI updates with the workflow name and execution ID. Both are **clickable links** that take you straight to the scene of the crime in n8n.

You never have to check the execution logs manually again. The logs will scream at you instead. 📉
