# 🔊 Faah Monitor — v1 (The Webhook Method)

> *"My workflows were failing silently. Now my whole house knows."*

v1 is the **OG method**. It requires no n8n API keys. Instead, you drop an HTTP Request node into your workflow's Error Trigger, and it sends a GET request to this server when things break.

If your n8n instance is on the cloud, you'll need a public URL (like ngrok) so n8n can reach your laptop. If they're on the same network, local IPs work fine.

---

## 🏃‍♂️ How to Run It

1. **Fork & Clone**
   ```bash
   git clone https://github.com/juppfy/faah-error-sound.git
   cd faah-error-sound
   ```

2. **Install & Start**
   ```bash
   npm install
   npm start
   ```

3. **Open the Dashboard**
   Go to **http://localhost:3001** in your browser.
   **Leave this tab open.** This is the tab that will scream at you.

---

## 🌩️ In the Cloud? You Need a Tunnel!

If your n8n instance is floating in the cloud somewhere (like n8n Cloud or a VPS) and your laptop is on your desk... n8n cannot hit `http://localhost`. It simply doesn't know where that is.

**Enter ngrok.** It gives your laptop a temporary public URL.

1. Install ngrok (`npm install -g ngrok` or download from their site)
2. Run this in a new terminal tab:
   ```bash
   ngrok http 3001
   ```
3. ngrok gives you a Forwarding URL that looks like: `https://8a7b-12-34-56-78.ngrok.app`

---

## 🤖 The n8n Setup

1. Open your workflow.
2. Add an **Error Trigger** node (if you want this to catch *any* error in the workflow).
3. Connect it to an **HTTP Request** node.
4. Set the HTTP Request node to:
   - **Method:** `GET`
   - **URL (Local n8n):** `http://[YOUR_LAPTOP_IP]:3001/error-sound`
   - **URL (Cloud n8n):** `https://[YOUR_NGROK_URL]/error-sound`
5. *(Optional Detail)* Pass the workflow name so the UI knows who to blame:
   ```
   https://[YOUR_NGROK_URL]/error-sound?workflow={{ $workflow.name }}
   ```

## 🤷‍♂️ Why use v1 instead of v2?
- You don't want to mess with API keys
- You only want the sound to play for *specific* workflows, not every single one on the instance
- You like setting up ngrok tunnels (who hurt you?)

Enjoy your newly found anxiety every time you hear a sound. 🎵
